# ESFD - Site Institucional Premium

Este projeto é um site institucional de alto padrão desenvolvido para a **ESFD (Empreiteira de Steel Frame e Drywall)** usando React, TypeScript, Framer Motion e CSS customizado.

## Tecnologias Utilizadas

- **React + Vito**: Framework frontend rápido e moderno.
- **TypeScript**: Tipagem estática para maior segurança e qualidade de código.
- **Framer Motion**: Biblioteca poderosa para animações complexas e performáticas.
- **Lucide React**: Ícones SVG leves e customizáveis.
- **CSS Modules / Variables**: Estilização modular e consistente com Design System.

## Estrutura do Projeto

- **src/components**: Componentes reutilizáveis (Hero, Services, About, Portfolio, etc.).
- **src/styles**: Estilos globais e tokens de design (cores, tipografia).
- **src/App.tsx**: Componente principal que integra as seções.

## Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse no navegador:
   `http://localhost:5173`

## Funcionalidades Premium

- **Parallax Scroll**: Efeito de profundidade na seção Hero.
- **Animações de Entrada**: Reveal suave de textos e elementos ao rolar a página.
- **Hover Effects**: Interações sofisticadas em botões e cards.
- **Menu Responsivo**: Navegação mobile fluida com animações.
- **Filtragem de Portfólio**: Transições suaves ao filtrar projetos por categoria.

## Personalização

- As cores e fontes globais podem ser alteradas em `src/index.css`.
- As imagens são placeholders do Unsplash/Imgur e devem ser substituídas pelas fotos reais dos projetos da ESFD.
